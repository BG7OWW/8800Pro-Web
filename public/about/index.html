森海克斯 8800Pro 的现代读写频工具

8800Pro 是为森海克斯 8800Pro 对讲机制作的多平台读写频工具。它把蓝牙、USB 写频线、信道编辑、频率快照、中继台库、FM 收音机、DTMF、VFO 和开机图等能力整理成一套更容易使用的控制台。

核心特性

围绕真实 8800Pro 手台的读写频流程设计，尽量把危险操作变得可见、可回退、可验证。

蓝牙与 USB 双链路

网页版支持蓝牙读写频和 USB 写频线。蓝牙连接时提供常规写频与快速写频两个入口，USB 写频线适合首次备份和稳定写入。

频率快照与备份

读频、写频和批量导入前都可以保存快照。配置出错时可恢复到之前状态，再重新写回设备。

全国中继台库

内置 HamCQ 中继台数据，按大区、省份和城市筛选，可把中继台参数快速写入当前信道。

完整信道管理

编辑接收/发射频率、亚音、名称、功率、带宽、扫描、繁忙锁、信令组等常用参数。

高级功能面板

提供 VFO、DTMF、FM 收音机、整机功能设置、卫星信道生成和开机图写入等页面。

读回校验思路

写频前自动备份，写频后建议再次读频确认，让浏览器操作尽量接近实体写频软件的可控性。

网页版

网页版运行在浏览器中，适合桌面端使用。Chrome / Edge 可以通过 Web Bluetooth 连接手台，也可以通过 Web Serial 连接 USB 写频线。未连接时页面只显示连接入口，连接成功后再进入控制台，避免新用户不知道从哪里开始。

  • 适合桌面端快速编辑、导入导出和批量整理信道。
  • 支持 JSON、Excel、CSV 等配置文件流转。
  • 服务器版本部署在 8800.743.world，GitHub Pages 版本也可作为备用访问入口。

手机版

手机版面向随身使用场景，适合在移动设备上查看配置、管理备份和执行读写频相关操作。由于 iOS 普通浏览器不开放本项目需要的 Web Bluetooth / Web Serial 能力，移动端建议使用原生 App。

为什么做这个项目

8800Pro 的读写频链路并没有完整公开文档。项目从社区开源实现、官方软件、Android APK、iOS 应用行为和真实手台回读中一点点对照协议，把信道、区域、VFO、DTMF、FM 和功能设置整理成可视化界面。

更低的使用门槛

把“先连接、先读频、再编辑、最后写回”的流程做成清晰路径，让第一次接触写频的用户也能理解每一步。

更强的可恢复性

频率快照、写前备份、导入导出和读回确认，都是为了减少配置丢失、误写和反复试错的成本。
